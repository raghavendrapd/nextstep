# NextStep AI Chrome Extension Icons

## Required Icon Sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Quick Icon Generator

1. Go to: https://favicon.io/icon-generator/
2. Or use: https://realfavicongenerator.net/

## Or Create Simple Icons:

1. Go to Canva.com
2. Create new design (custom size: 128x128)
3. Add text "NS" or lightning bolt
4. Use teal color (#50eede) on dark background (#131315)
5. Download as PNG

## Alternative - Use Online Converter:

1. Create icon at: https://www.figma.com/
2. Export as PNG
3. Resize to required dimensions

## For Development - You Can Use These:

For now, you can use any 16x16, 48x48, and 128x128 PNG images. 
The extension will work without perfect icons.
